This repository contains two datasets used to demonstrate supervised and unsupervised 
machine learning techniques.

The Predict Students Dropout and Academic Success Dataset is used for supervised learning, 
where the objective is to predict students’ academic outcomes (dropout, enrolled, or graduate) based on demographic, academic, and socioeconomic factors. In contrast, the Wholesale Customers Dataset is used for unsupervised learning, focusing on discovering patterns in customer purchasing behavior using annual spending data across different product categories. Together, these datasets provide practical examples of applying machine 
learning models for prediction and pattern discovery.
